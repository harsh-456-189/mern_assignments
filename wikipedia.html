Overview

The HyperText Markup Language, or HTML, is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content. HTML is often assisted by technologies such as CSS and JavaScript to format and control behavior, but HTML itself is used to define the structure of a web page.

History

HTML was first proposed by Tim Berners-Lee in 1993 as a way to share documents via the World Wide Web. Early versions were simple and primarily focused on text and links. Over time, HTML evolved through various versions, with HTML4 being widely adopted in the 1990s and HTML5 becoming the modern standard in the 2010s, supporting multimedia, accessibility, and responsive design.

Markup

HTML uses markup symbols or tags to describe content. For example, paragraphs are marked with <p>, headings with <h1> to <h6>, and links with <a>. Tags usually come in pairs, with opening and closing tags, and can contain attributes like href or src to control functionality.

Elements

An HTML document is made up of a tree of HTML elements. Each element can contain other elements, text, or both. Below are some common ones:

  • <h1> to <h6> — Headings of different levels
  • <p> — Paragraph
  • <a href=""> — Hyperlink
  • <img src=""> — Image
  • <table>, <tr>, <td> — Tables and rows
